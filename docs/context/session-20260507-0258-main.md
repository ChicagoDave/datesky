### 5. Process Failure Discovered and Corrected

Throughout the session I had been treating HTTP 000 responses as "domain not registered." This is incorrect — HTTP 000 means no HTTPS server responded, not that no domain registration exists. The distinction matters: a parked domain with no web server returns HTTP 000 but is very much registered.

User caught this when nomare.com turned out to be registered (since 2007, Netherlands registrant via Easyhosting, dormant Dutch parked page for 18+ years).

Whois verification then confirmed: nomare.com is registered and dormant. .app, .so, .io, .co, .net all returned no NS records — genuinely available. I acknowledged the process error to the user.

**Going forward**: whois is the authoritative check for domain availability. HTTP probes only test whether a server responds.

## Key Decisions

### 1. Trademark-First Naming Workflow

Never get attached to a candidate name before checking USPTO + operating brands + cultural overlap. The Pavilion failure (registered domain, built mockup, then discovered conflict) and this session's Loka/Stoa/Vellir failures (developed attachment, then encountered the block) establish the canonical process to avoid. ADR-0002 codifies this as a constraint on all future naming sessions.

### 2. Brand Name: Nomare (no-MAR-ay)

Archaic Italian for "to call by name." Generated by a custom phonotactic script (`scripts/name-gen.py`) after metaphor-anchored naming proved unable to clear trademark. The hidden Italian meaning is on-theme and teachable via onboarding copy without being obvious enough to attract competitors. See ADR-0002.

### 3. Primary Domain: nomare.net

.net chosen over .app for anti-precious positioning — open-internet, pre-platform-era connotation matches the product's bus-stop/public-space tone better than .app's Silicon Valley product-launch signal. nomare.com is dormant (Netherlands, 2007) and not pursued at this time. See ADR-0002.

### 4. Soft Consonant Palette for Naming

M, N, S, L, H, R, G, T, W produces brand-feel that is warmer and less aggressive than the full English consonant set. User-driven aesthetic call, now encoded in `scripts/name-gen.py` for any future naming needs on this or related projects.

### 5. HTTP 000 Is Not "Unregistered"

A process discipline correction. HTTP 000 = no HTTPS server responded. Whois is the authoritative check for registration status. This mistake was made throughout the session and caught by the user; acknowledged and corrected before the final domain decision.
