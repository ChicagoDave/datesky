## Key Decisions

### 1. Dual-namespace deletion failure semantics (ADR-0004)
Delete both `app.nomare.profile/self` and `app.datesky.profile/self`. `RecordNotFound` on either is silenced (expected for users who joined after dual-publish launched). Any other PDS error aborts the entire operation so the user can retry. Local cleanup runs only after both PDS calls converge, making retries idempotent. This honors the about-page promise of full record removal.

### 2. Typed confirmation as the project-wide convention for irreversible actions (ADR-0005)
Typed `DELETE` (case-insensitive after trim/uppercase) is now the mandatory UX gate for any irreversible destructive action in Nomare. The API does not validate the typed string — friction is a UI guard against accidents, not an auth control. This decision constrains future sessions: any destructive action added later must follow this pattern.

### 3. Hard delete for v1, no soft-delete machinery permitted (ADR-0006)
No `deleted_at` columns are added. The about-page copy at `src/app/about/page.tsx:71` is treated as load-bearing. Soft delete would require schema migration, a cleanup job, and browse-page filtering — substantial scope. A future ADR may revisit if accidental-deletion incidents emerge at scale.

### 4. Belt-and-braces local delete (Decision 2, no ADR needed)
`deleteProfile(did)` and `deleteOAuthSession(did)` are called synchronously in the API handler in addition to the Jetstream delete event handler that already covers them asynchronously. Duplicate deletes are idempotent. This eliminates the stale-profile-flash race condition where a user could briefly see their own profile in browse after deletion.

### 5. Hard navigation on deletion success (implementation detail)
`window.location.href = "/goodbye"` rather than `router.push("/goodbye")`. Next.js `router.push` performs a client-side navigation that may keep stale cookie state in the browser's in-memory session. Hard navigation forces a full round-trip and ensures the destroyed iron-session cookie is not replayed by the browser cache.
